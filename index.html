How to use this

  • Loose bananas priced per kg: set type to "per kg", enter weight in grams, enter price
  • Bundled bananas at a fixed price: set type to "per item/bundle", enter weight of the whole bundle in grams
  • Everything is normalised to cost per 100g so you can compare directly
  • Weigh bundles using the supermarket's produce scale — it's free!
